A Breif Overview

For this project the overall goal was to take a university calenber app and redesign it so that it was easier for the students to turn in and find there assignments. Some of the most important features that needed to be updated was to make a way for the students to view the details of there assignments from the Calender page and to be able to submit certain assignments on the page as well.

My Role

On this project I was to create the new UI for the sight. As well as prototype the app and conduct user testing on it as well. The first step on the process was to simplify the design and come up with a stragy to make the app more user friendly.

Visual Exploration

First I start out with some initial planning and deciding where I would like the information to be laid out throughout the site. I will need to decide where I want to lay out the images, text, and other information required for the site. I then laid out the same designs for mobile devices.

logo design and color scheme

After deciding on the layout of the webpage I want to pick a color scheme that can be used for the website. I choose to go with a handful of neutral colors that could be used for the site. I also to some time to experiment with some possible logo designs if my client would like to use them.

Final Mockup

I later created the final UI for the website using the images, text and color chooses. I planned the image gallery layout with the images as seen in the final UI before hand. I wanted the gallery page design to look seemless and the images to all be diffrent sizes. this is what thet final layout of the webaite will look like. the last step now is to prototype the artboards in a prototyping software, in this case I will be using Abode XD.
